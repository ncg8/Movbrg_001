Melodyz

Instant Melody Creator.
Melody Scores for composition, improvisation and instrument practice.

Melodyz generates endless melody scores for composition, improvisation and instrument practice for classical music or any other style.

Choosing from C, G and F clefs, Melodyz generates score melodies for a wide range of instruments practice and music composition applications.

With Melodyz you can easily create a entire staff line melody by pressing on each of the black buttons on each screen, or, you can create melodies from scratch by pressing on each individual note.

Practice instrument with the metronome included. Set the tempo and swipe through the screens playing the melodies generated on the staff.

All the following features can be used at the same time, each oneindividually. That way it is possible to have six different generators screens in the same song, multiplying the number of possible melody ideas to use.

CreativJam

From musicians to musicians,
break the writers block and the old creation patterns with CreativJam, and get endless ideas for your songs..

CreativJam is an app that generates endless song ideas, hooks, unusual note sequences, improvisation chord progressions, standard chord progressions and scale wide modulations, focus in breaking the usual patterns of the song creation process. As well as a lyric writer and a metronome to test the new ideas immediately.

With CreativJam you easily can create a song from scratch, or you can start with your own song, and over that,generate new ideas and note solutions that can lead your song creation to paths that you never thought before.

Although the CreativJam generation algorithms are designed directed to music styles such as R&B, Soul, Funk, Disco, Latin and Singing music, it can be used in any other style.

All the following features can be used at the same time, each one generating individually. That way it is possible to have five different generators in the same song, multiplying the number of possible note ideas to be used.
